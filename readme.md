# Chat en tiempo real usando Angular y .NET

## Información

Este proyecto está pensado para aprender las bases de la comunicación entre una página web y un servidor usando como ejemplo un chat en tiempo real.

## Objetivos

- Crear un sistema de inicio de sesión
- Configurar SQLite con Entity Framework
- Proteger contraseñas de usuarios
- Proteger endpoints con JWT
- Mejorar la seguridad de los tokens JWT
- Crear una sala de chat
- Implementar WebSockets
- Proteger conexiones WebSocket

## Tecnologias utilizadas

- [Angular 21](https://angular.dev)
- [.NET 10](https://dotnet.microsoft.com/es-es/download/dotnet/10.0)
- [Tailwindcss 4.2](https://tailwindcss.com/)
